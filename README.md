# Brute-Force-senhas
Simulando com o Kali Linux um ataque de Brute Force, utlizando a ferramenta Medusa em conjunto ambientes vulneráveis do Metasploitable.

Ambiente virtual (Virtual Box - Oracle) com duas maquinas, Kali linux e Metasploitable, cujo o ambiente há varias vulnerábilidades para estudo.
Ambas configuradas para mesma na mesma rede exclusiva do hospedeiro (host-only)

No ambiente Kali Linux realizei uma invasão de login e senha no metasploitable, realizando Brute force no protocolo FTP através do MEDUSA
